Task 03

Аналіз замовлень (JOIN)

SELECT orders.order_id,
       users.first_name, users.last_name,
       orders.status
FROM `bigquery-public-data.thelook_ecommerce.orders` AS orders
JOIN `bigquery-public-data.thelook_ecommerce.users` AS users
  ON orders.user_id = users.id
WHERE orders.status = 'Shipped';
// результат виконання запиту 03
Task 04

Фінанси — найдорожчі замовлення

SELECT order_id, user_id,
       ROUND(SUM(sale_price), 2) AS sale_price
FROM `bigquery-public-data.thelook_ecommerce.order_items`
GROUP BY order_id, user_id
ORDER BY sale_price DESC;
// результат виконання запиту 04
Task 05

Географія покупців

SELECT country,
       COUNT(id) AS users_amount
FROM `bigquery-public-data.thelook_ecommerce.users`
GROUP BY country
HAVING COUNT(id) > 500
ORDER BY users_amount DESC;

03

Моделювання (Linear Regression)

Прогнозування суми продажів (Gross Amount) на основі вікової групи покупця за допомогою лінійної регресії з OneHotEncoding.

X = df[["Age Group"]]
y = df["Gross Amount"]

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=42
)

preprocess = ColumnTransformer(
    transformers=[
        ("cat", OneHotEncoder(handle_unknown="ignore"), ["Age Group"])
    ]
)
lr = Pipeline(steps=[
    ("preprocess", preprocess),
    ("model", LinearRegression())
])

lr.fit(X_train, y_train)
y_pred = lr.predict(X_test)

feature_names = lr.named_steps["preprocess"] \
    .named_transformers_["cat"] \
    .get_feature_names_out(["Age Group"])

coefficients = lr.named_steps["model"].coef_

for name, coef in zip(feature_names, coefficients):
    print(name, coef)
